## 3. リアルタイムスキャン(かざしっぱなし)の実現方法

NORIの構想「スキャンしながらリアルタイムで単語と発音がすぐ出る」は**今日の技術で実現可能**。2つの選択肢:

### 候補A: Gemini Live API(本命)
- `gemini-3.1-flash-live-preview`: カメラ映像を**1fpsでストリーミング**しながら、音声で即座に答えるAPI。映像+音声+テキストの双方向リアルタイム
- 価格: 入力$0.75/出力$4.50(百万トークン)。映像は258トークン/秒
- **コスト試算**: 映像1分 ≈ 15,500トークン + 音声応答 + ターン毎にセッション文脈を再課金 → **1分あたり約2〜10円**。かざしっぱなし10分で数十円〜百円
- 判定: 常時オンは無料ユーザーには出せない。**Pro限定機能「ライブスキャン」**として最適。現行の「1枚スキャン」方式(1回0.1円台)は無料枠として維持

### 候補B: OpenAI Realtime API(gpt-realtime-2.1、2026-07-06リリース)
- 音声会話のリアルタイム性は最高(p95レイテンシ25%改善)。**画像入力は対応、映像ストリーミングは未対応** → かざしっぱなしには不向き
- 使いどころ: **会話練習モード**(ネイティブと話す練習をリアルタイム音声で)の将来機能。GPT-Realtime-Translate(70言語→13言語の同時通訳)も旅行者向け機能の種

### 実行計画(憲法: 小さく試す)
1. βテスト完了後、Gemini Live APIで**3分間のプロトタイプ**(かざす→音声で単語+発音が返る)を別ページ(/live、Pro+admin限定)に実装
2. 実測コスト/分を usage_events に記録し、Pro価格(NT$149)との整合を確認
3. 合格ならPhase Cの目玉機能「ライブスキャン(Pro)」として正式化

## 4. コーパス(COCT)の商用利用

国教院の華語文語料庫(coct.naer.edu.tw)は頻度表・分詞システム等を公開しているが、**商用利用の可否は公開ページで明示されていない** → 国家教育研究院に利用許諾を問い合わせるのが正攻法(NORIのタスク: 問い合わせフォームから「商用アプリでの詞頻統計の利用」を照会)。回答までは現行のAI推定頻度(カードの「頻度・どこで使う」欄)で運用。

## 5. モデル更新の運用ルール

- モデル名は全てenv(Secrets)で切替可能 — コード変更なしで最新モデルへ移行できる構成を維持する
- 新モデルへの切替は必ず「3語でテスト→耳と目で確認→本番」(憲法§2)
- 料金は毎月変わる。四半期ごとにこのドキュメントを更新する
